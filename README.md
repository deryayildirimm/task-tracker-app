# 🚀 Task Tracker App

Modern **React** kullanılarak geliştirilmiş basit ve kullanıcı dostu bir görev yönetim (Todo) uygulamasıdır.
Kullanıcılar görev ekleyebilir, listeleyebilir, güncelleyebilir ve silebilir.

---

## ✨ Özellikler

* ➕ Görev ekleme
* 📋 Görev listeleme
* ✏️ Görev güncelleme
* 🗑️ Görev silme
* 💾 LocalStorage ile veri saklama
* ⚠️ Form validasyonu ve hata yönetimi
* 🧪 Vitest ile yazılmış unit testler
* ⚙️ GitHub Actions ile CI pipeline

---

## 🛠️ Kullanılan Teknolojiler

* React
* Vite
* Bootstrap 5
* JavaScript (ES6+)
* Vitest (Test)
* GitHub Actions (CI/CD)

---

## ⚙️ Kurulum

Projeyi klonlayın:

```bash
git clone https://github.com/KULLANICI_ADIN/task-tracker-app.git
cd task-tracker-app
```

Bağımlılıkları yükleyin:

```bash
npm install
```

Projeyi başlatın:

```bash
npm run dev
```

---

## 🧪 Test

```bash
npm run test
```
